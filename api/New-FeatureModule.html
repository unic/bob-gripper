New-FeatureModule

Creates new projects for a new feature module according to our architecture guidelines.

Syntax

New-FeatureModule [<CommonParameters>]

Description

Creates new projects for a new feature module according to our architecture guidelines. You will be asked a few questions and Gripper will create new projects in proper directories in the file system and Visual Studio solution.
